Windows系统添加Ubuntu20.04（双系统，单磁盘）

步骤一：准备 Windows 11 环境 & USB驱动器

  1. 关闭bitLocker
    • Windows + R打开「运行」对话框，输入msinfo32，按Ctrl + Shift + Enter以管理员权限打开「系统信息」工具，查看「设备加密支持」条目，如果是“满足先决条件”说明当前设备支持 BitLocker 设备加密
    • 「设置」>「隐私和安全性」>「设备加密」> 关闭
  2. 关闭快速启动
    • Windows + R打开「运行」对话框，输入control panel，点击Enter打开「控制面板」
    • 「硬件和声音」——「电源选项」——「选择电源按钮的功能」——「更改不可用的设置」——「启用快速启动」（关闭此选项）——「保存更改」
  3. 禁用安全启动
    • 重新启动计算机并进入 BIOS 或 UEFI 设置，通常在启动过程中按下F2、F10、Delete等键。
    • 前往「安全」或「启动」选项卡
    • 找到「安全启动」选项并将其设置为已禁用
    • 退出 BIOS/UEFI 设置之前保存更改
  4. 为Linux分配空间
    • 在此电脑右键 管理 — 磁盘管理，在想要分出空间的部分右键压缩卷，输入分配给Ubuntu的空间大小后压缩
  5. 获取 Linux 发行版，制作可启动的 USB 驱动器
    • 前往 Ubuntu 官方网站下载最新的 ISO 文件
    • rufus制作ubuntu启动盘 烧录系统会格式化U盘
    • Ventoy制作U盘启动系统盘 允许多个ios在U盘中，安装Ventory会格式化U盘

步骤二：安装驱动器

  1. 将可启动的 USB 驱动器插入您的系统。重启机器进入BIOS，利用F6,F7调整启动顺序，将USB启动调整到第一个，保存并退出。
  2. 「Install Ubuntu」>「keyboard Layout」默认英文>「Wireless」不用联网>「Updates and other software」默认Normal >「Install type」Something else>「location」随意>「个人信息自己填」>安装
  3. 「Install type」
    • 选择从Windows中划分出来的那部分（从大小判断）
    • 分区 /boot: 1G ext4格式
    • 分区 swap: 4G swap area格式 （交换空间：交换分区相当于Windows中的“虚拟内存”，如果内存低的话（1-4G），物理内存的两倍，高点的话（8-16G）要么等于物理内存，要么物理内存+2g左右）
    • 分区 / : 20G ext4格式
    • 分区 /home: 剩余所有 ext4格式
    • 下方的安装启动引导器的设备选择 /boot的path

步骤三：验证双启动配置

  1. 系统安装很快，如果等了很久还没安好可能是分盘出了问题
  2. 安装好之后出现提示单击“立即重启”，按照指示移除 USB 驱动器并按Enter
  3. 安装后，您的计算机应该显示一个启动菜单（GRUB），让您在 Ubuntu 和 Windows 11 之间进行选择。当 GRUB 菜单出现时，选择Ubuntu并点击Enter，或选择Windows并点击Enter。
