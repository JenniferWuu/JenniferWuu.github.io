What is Grasshopper?

Rhino 建筑建模软件下的一个可视化编程插件。

How to use Grasshopper?

参数化建模 ：使用算法建模

算法 ：通过一些简单指令，完成复杂任务。

输入—指令—输出

输入什么？

  • 几何形：点、线、面

  • 数字：数量、长度、角度、颜色…

  • 位置：坐标系、方向

  • 列表panel：批量输入

什么指令？
  • Maths: domain、Matrix、 .etc
  • Sets: List、.etc (一般处理列表)
  • Vector: Plane(坐标系)
  • 线面操作：Curve,Surface
  • Transform: 缩放、镜像、移动…
  • Display
