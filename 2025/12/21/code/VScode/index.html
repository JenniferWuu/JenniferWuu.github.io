使用 .vsix 下载VScode的jupyter extention

  1. 进入VScode的插件官网，搜索 jupyter，双击进入

  2. 在 Overview 界面右下角有 More Info，可以看见：

Version Unique Identifier
2025.10.2025121801 ms-toolsai.jupyter

vscode下载插件的链接格式为：[https://marketplace.visualstudio.com/_apis/public/gallery/publishers/{publisher_id}/vsextensions/{extension_name}/{version}/vspackage]

替换链接中的占位符后：[https://marketplace.visualstudio.com/_apis/public/gallery/publishers/ms-toolsai/vsextensions/jupyter/2025.10.2025121801/vspackage]，下载即可

  1. 在 VScode Extensions 中选择 Install from VSIX...，定位到下载文件的位置即可

  2. 注意！在 VScode 中点击 Help - About 可以查看版本 Date，插件 release 时间要稍早于这个 Date

    • 可以通过查看插件官网的 VersionHistory 界面查看历史版本
    • 如果需要更早的，就需要进入Overview界面-Project Details下面的Github寻找
      • Github在main分支下选择tag - view all tag，选择 release，根据 Assets 中的日期去定位（很奇怪jupyter这里没有.vsix，我是根据这个日期前后调整下载链接中的日期找到了一个）
