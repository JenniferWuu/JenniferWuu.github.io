Server连接与性能查看

Created2023-08-12|Updated2026-03-10|linux
|Post View:

什么是服务器？一直开着、性能更强、专门用来存储数据或运行服务，并通过网络响应其他电脑（客户端）请求的计算机（有 CPU、内存、硬盘和操作系统）。

连接服务器

  1. 命令行ssh连接：ssh 用户名@IP_address，根据指令输入密码（密码输入是不可见的），回车，完成。

  2. 命令行利用scp命令从服务器下载文件：scp -r ironman@10.233.177.202:home/ironman/xxx/ /home/myname/xxx/

  3. 终端软件

    • Xshell + Xftp

    • MobaXterm

      • 可以split分屏

  • FileZilla【文件 - 站点管理器】
